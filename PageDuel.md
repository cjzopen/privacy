感謝您使用 PageDuel-beta（以下簡稱「本套件」）。本套件利用 Gemini AI 分析競品視覺權重並生成戰略報表。為了保護您的隱私，我們在此說明本套件如何處理資訊。

# 1. 隱私核心原則

本套件以「透明」與「安全」為設計核心。我們保證：

 - 完全不存取使用者資料：本套件沒有後端伺服器，不會上傳您的個人資訊。

 - 不引用第三方程式：除了必要的 Google Gemini AI API，本套件不引用任何外部腳本或程式。

# 2. 如何處理資料

## 抓取網頁內容

為了確保安全性與技術透明，本套件不連線至其他網站解析內容，也不會自動爬取。

 - 手動抓取：也因為如此，本套件無法靠輸入網址來爬取內容。使用者須在目標網站上，利用本工具提供的公開程式碼抓取網頁內容。

 - 處理範圍：本套件僅處理您上傳的檔案、手動輸入的設定或選定的網頁，且流程完全在您的瀏覽器內完成，不會有第三方知道使用者處理了什麼資訊（Google Gemini AI API 除外）。

## 使用 Gemini AI 分析

本套件將抓取的內容發送至 Google Gemini 模型。

 - API 金鑰 (API Key)：本套件目前為免費 Beta 版本，使用者須輸入自己擁有的 API Key。

 - 輸入金鑰：您的 API Key 只可能會儲存在您電腦的瀏覽器本地空間（若有自行記憶），開發者無法得知或存取該金鑰。

# 3. 儲存與傳輸資料

 - 本地處理：所有分析與結果都在您的瀏覽器內完成。

 - 無外部連線：除了向 Google Gemini API 發送請求，本套件不將任何資料傳輸至其他第三方網站。

# 4. 您的權利與控制

清除資料：您可以隨時在套件設定中清除分析紀錄。

移除套件：解除安裝套件後，瀏覽器會移除所有儲存在本地的設定。

若套件更新功能，我們可能會修訂此隱私權聲明。請您定期查看。
