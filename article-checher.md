# 隱私權聲明與權限說明

本擴充功能（Article Checker Beta）重視您的隱私，所有資料處理邏輯均遵循以下原則：

## 1. 隱私權聲明
* **資料不落地**：本擴充功能不會收集、儲存或傳送您的任何個人資訊至開發者的伺服器。
* **API 溝通**：您提供的網頁內容與 API Key 會直接傳送到 Google Gemini API 進行處理。相關資料的使用受 [Google 隱私權政策](https://policies.google.com/privacy) 規範。
* **本地儲存**：您的 API Key 與設定僅儲存在您瀏覽器的本地儲存空間（Chrome Storage），不會同步至雲端或任何其他地方。

## 2. 權限用途說明
本擴充功能要求以下權限以確保核心功能運算：

| 權限項目 | 用途說明 |
| :--- | :--- |
| **storage** | 儲存您的 Gemini API Key、偏好的模型設定、思考程度（Thinking Level）以及最後一次的分析結果。 |
| **activeTab** | 當您啟動「選取網頁元素」時，取得當前分頁的暫時性存取權限，以便讀取您所選取的區塊內容。 |
| **scripting** | 用於在網頁中注入內容選取工具（Selection Overlay），讓您能直接在頁面上透過點擊來選擇要分析的區塊。 |
| **tabs** | 當分析完成後，系統需要此權限來自動開啟新的分頁以展示詳細的分析對照表與優化建議。 |
